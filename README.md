# nvim-config
## requirement
- Vim Plugin
- Node
- Ruby

## Install
1. `PlugInstall`
2. `CocInstall coc-solargraph`
3. `npm install -g tree-sitter-cli`
4. `TSInstall ruby`

## TroublesShooting
1. Language Server 在 Windows 中要下此指令 `powershell -executionpolicy bypass -File install.ps1`，參考此[連結](https://github.com/autozimu/LanguageClient-neovim/blob/next/INSTALL.md#with-a-plugin-manager)
